GitHub Pages = Mode autonome uniquement
Sur GitHub Pages, l'Add-in fonctionne en mode JS pur — tous les calculs (PCA, PLS, K-Means, Mahalanobis) s'exécutent directement dans le navigateur via chemMath.js. Le backend Python (FastAPI + scikit-learn) est optionnel et nécessite un serveur séparé. Pour activer Isolation Forest, LOF, SVM ou Random Forest : déployer le backend sur Railway ou Render →

Mise en place

Installation en 5 étapes

L'Add-in fonctionne directement depuis GitHub Pages dans un navigateur. Pour l'intégrer dans Excel Desktop, suivez les étapes ci-dessous.

Étape 01
Cloner le dépôt

Clonez le projet et naviguez dans le dossier.

git clone https://github.com/ VOTRE-USERNAME/chemai-addin cd chemai-addin
Étape 02
Installer Node.js

Installez les dépendances et générez les certificats SSL locaux.

npm install npx office-addin-dev-certs install
Étape 03
Lancer le serveur local

Démarrez webpack-dev-server sur HTTPS (requis par Office.js).

npm start # → https://localhost:3000
Étape 04
Charger dans Excel

Dans Excel : Insérer → Compléments → Mes compléments → sélectionner manifest.xml

# Excel Desktop Insérer → Compléments Office → Télécharger un complément → Sélectionner manifest.xml
Étape 05 (optionnel)
Backend Python

Pour activer les algorithmes avancés (Isolation Forest, SVM, RF…)

cd backend pip install -r requirements.txt python main.py

Algorithmes implémentés

Chaque algorithme est documenté avec sa référence originale et ses métriques de sortie.

Module Algorithme Mode JS Métriques Référence
PCA SVD (LAPACK) + déflation itérative ✓ Natif Valeurs propres, variance expliquée, T², Q Jolliffe (2002)
PLS-1 NIPALS (Nonlinear Iterative PLS) ✓ Natif R²cal, R²cv, RMSEC, RMSECV, VIP Wold et al. (2001)
K-Means K-Means++ (initialisation optimale) ✓ Natif Inertie, Silhouette, Calinski-Harabász Arthur & Vassilvitskii (2007)
Clustering hiérarchique Ward, Complete, Average, Single ✓ Natif Linkage matrix, dendrogramme Ward (1963)
Mahalanobis Distance Chi² multivariée ✓ Natif Score par échantillon, seuil Chi²(n, α) Mahalanobis (1936)
T² de Hotelling Statistique F dans l'espace PCA ✓ Natif T² individuel, limite F-distribution Hotelling (1931)
Q résiduel Jackson-Mudholkar (1979) ✓ Natif Q par échantillon, seuil JM Jackson & Mudholkar (1979)
Isolation Forest Liu et al. — arbres d'isolation Backend Python Score d'anomalie, contamination Liu et al. (2008)
LOF Local Outlier Factor Backend Python LOF score, voisins k Breunig et al. (2000)
LDA Analyse Discriminante Linéaire Backend Python Accuracy, matrice de confusion, F-importance Fisher (1936)
Random Forest Ensemble de 100 arbres de décision Backend Python Accuracy CV, feature importance, OOB Breiman (2001)
